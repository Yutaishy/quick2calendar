# デザインUX仕様書 (v3.0)

本ドキュメントは、Quick2Calendar のUX/UIを **ChatGPT Desktop App (macOS) の「Option + Space」ランチャー体験** に揃えるための一次仕様です。  
クイック入力と設定画面は **同一のガラス板テーマ** を共有します。

## 1. クイック入力（Quick）

### 1.1 見た目（Glass / One Sheet）

- **形状**: 検索バー型（角丸 16px）
- **質感**: 透明な黒のガラス板（Blur 40px / Saturate 180%）
- **背景**: 単色ではなく微小グラデーションで光の当たり方を表現（暗色のみ）
- **縁（エッジ）**: 上辺を少し明るく、下辺を暗くして「板の厚み」を出す
- **一枚板**: 外枠と中身の間に「別の板」に見えるレイヤーを作らない（ズレて見せない）

### 1.2 レイアウト

- **空状態**: 1行Composer
  - 左: 添付（クリップ）アイコン
  - 中央: テキスト入力（placeholder: `質問してみましょう`）
  - 右: 送信ボタン
- **履歴あり**: 上部に会話リストを表示し、下部にComposerを固定
- **ステータス**: 添付件数・エラー等はComposer直下に最小限で表示
- **高さ**: 内容に追従して自動調整し、不要な余白を作らない（最大 600px）

### 1.3 操作（ショートカット）

- **起動**: グローバルショートカット（例: Option + Space）
- **送信**: `Cmd + Enter`（送信ボタンでも可）
- **改行**: `Enter` / `Shift + Enter`
- **閉じる**: フォーカスが外れたら自動で非表示（Hide on Blur）、または `Escape`
- **新しいチャット**: `Cmd + Shift + O`（会話/聞き返し状態/添付をリセットし、折りたたみ状態へ）

### 1.4 ドラッグ

- **背景領域はどこでもドラッグ可能**
- 入力欄/ボタン/リンクなどインタラクティブ要素はドラッグ不可（操作性優先）

## 2. 設定画面（Settings）

### 2.1 見た目

- クイック入力と同一のガラス板テーマ（背景・縁・質感を統一）
- セクションは薄いパネルで区切る（背景を透かす）

### 2.2 スクロール & ドラッグ

- 内容は縦スクロール可能
- ドラッグは **上部 32px** のみ（スクロール操作を奪わないため）
